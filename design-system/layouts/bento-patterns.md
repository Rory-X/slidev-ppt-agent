# Bento Grid Layout Patterns

Canvas: 980 Г— 552 px | Safe area: 900 Г— 472 px (40 px margins all sides)
Gap between cards: minimum 16 px, recommended 20вҖ“24 px

---

## Global Rules

- **One visual heavyweight per slide** вҖ” code block OR diagram OR metric table OR image. Never two heavyweights competing.
- **Hierarchy through size** вҖ” primary card вүҘ 60 % of safe area, secondary вүӨ 40 %.
- **Whitespace is a design element** вҖ” never fill every pixel. Leave breathing room between cards and around the slide edges.
- **Consistent gap** вҖ” pick one gap value (20 px recommended) and use it throughout the deck.
- **Alignment** вҖ” all card edges align to a shared grid. No arbitrary offsets.
